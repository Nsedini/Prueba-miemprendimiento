HTML y CSS para principiantes

HTML y CSS son dos herramientas importantes para dar inicio en el mundo del desarrollo web. Por lo tanto, en este curso aprenderas todo sobre HTML y CSS, desde lo mas básico hasta un nivel intermedio, logrando desarrollar tu primera página web responsiva, sin la necesidad de aplicar librerías externas para CSS.

Estilos con SASS

Sass es una herramienta importante hoy en día para todo desarrollador web, ya que permite crear estilos más ordenados, eficientes y con menos lineas de codigo. Por ello, en este curso aprenderas todo lo necesario para crear hojas de estilos desde la sintaxis de SCSS , implementando tecnicas propias de SASS.
